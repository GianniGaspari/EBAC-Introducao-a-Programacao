Matrizes

Módulo 11

As matrizes são estruturas de dados utilizadas em computação para armazenar coleções de números ou valores. Elas são compostas por linhas e colunas, sendo que cada elemento da matriz é identificado por sua posição (linha, coluna). Matrizes são amplamente utilizadas em áreas como processamento de imagens, processamento de sinais e cálculo numérico. Algumas operações comuns com matrizes incluem adição, subtração, multiplicação e inversão. A biblioteca de álgebra linear (Linear Algebra Library) é comumente usada para implementar essas operações em linguagens de programação como Python, C++ e Java.

  • Armazena um conjunto de informações do mesmo tipo.
  • Realiza operações isoladas com seus valores.
  • Assim como os vetores, os índices das matrizes começam no zero(0).
  • Vetor → array de uma dimensão.
  • Matriz → array de duas ou mais dimensões.
